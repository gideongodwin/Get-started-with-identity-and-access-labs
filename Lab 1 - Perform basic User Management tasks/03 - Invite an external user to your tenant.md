## Task 3 - Invite an external user to your tenant

1. Sign in to the [Microsoft Entra Admin Center](https://entra.microsoft.com)

2. Select Users > new user > create new user > invite external user

   <img width="1413" height="329" alt="517728223-d89ce541-8ba1-4de0-8aec-9ad801e02e36" src="https://github.com/user-attachments/assets/72b5c613-b5b8-47ad-81b2-3d8cd5ddadaa" />

3. Enter the information for your new external user:

    | Field        | Value                                                                                                             |
    |--------------|-------------------------------------------------------------------------------------------------------------------|
    | Email        | ExtUser@testemail.com                                                                                             |
    | Display name | External User                                                                                                     |
    | Message      | Thank you for joining the company for this short work project. We look forward to building this project together. |

    <img width="1241" height="828" alt="520414141-f5872f01-73a7-463f-9d9e-2b7ad17ea445" src="https://github.com/user-attachments/assets/d6e0d200-f557-4f8a-97a4-06359386b56a" />
 
4. Select the `Review + invite`, then invite
