## Task 1 - Create a new user

1. Sign in to the [Microsoft Entra Admin Center](https://entra.microsoft.com)

2. From the menu on the left, select User and then All users

3. Select New user > Create new user.

   <img width="568" height="274" alt="create-new-user-menu" src="https://github.com/user-attachments/assets/27e3ba8e-1e48-4987-bbed-a4b7d9103d81" />

4. Enter the requested information:
   
   | Field               | Value            |
   |---------------------|------------------|
   | User principal name | BhogeswarK       |
   | Display name        | Bhogeswar Kalita |

   <img width="962" height="588" alt="520412943-88968803-1ec8-4394-b35b-b4b4961bc93a" src="https://github.com/user-attachments/assets/16ded64e-2ccf-40f3-8e6d-4100b67f46f2" />
 
5. Leave the Mail nickname as is.

6. Make a copy of the Password (auto-generated) and store it in Notepad. You will need it later.

7. Leave Account enabled checked.

8. Select the Properties tab.

9. Set the values for first and last name:
    
   | Field               | Value            |
   |---------------------|------------------|
   | User principal name | BhogeswarK       |
   | Display name        | Bhogeswar Kalita |

   <img width="1153" height="352" alt="517725755-80815abe-dc6a-466b-a83f-9e882176289a" src="https://github.com/user-attachments/assets/0b1d2d33-e6c9-4993-b262-6575b3535776" />

   > Note that you can set the User type to `Member` If you pick `Guest`, it will restrict the users capabilities.

10. Scroll down to the bottom of the page.

11. Set the Usage location to United States or to whatever region you are in.

<img width="1155" height="804" alt="517725794-893ae4b7-2e5d-44b5-af33-42b758be4183" src="https://github.com/user-attachments/assets/0ba1ec3e-65c5-4604-bc62-d09e66e03f70" />
 
12. Select Review + create, then Create
