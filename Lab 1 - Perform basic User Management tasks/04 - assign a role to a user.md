### Task 4 - Assign a role to a user

1. Open the Microsoft Entra admin center at https://entra.microsoft.com

2. If prompted, log in using the credentials for your tenant.

3. Select Users > All users from the left menu.

4. Select the user Bhogeswar Kalita.

<img width="1423" height="505" alt="12" src="https://github.com/user-attachments/assets/82fed3e4-42d3-4240-bc72-80585080a84a" />

5. From the newly opened menu select Assigned roles.

6. At the top of the screen, select + Add assignment.

7. From the Directory Roles, choose the Attribute Definition Reader role
