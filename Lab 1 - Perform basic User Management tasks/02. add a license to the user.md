### Task 2 - Add a license to the user

1. Open the Microsoft 365 admin center at https://admin.microsoft.com 

2. If prompted, log in using the credentials for your tenant.

3. Select Billing from the menu on the left side of the screen.

4. Select Licenses from the list.

5. When the list of licenses appears, select Power Automate Premium.
