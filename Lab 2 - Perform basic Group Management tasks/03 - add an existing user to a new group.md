## Task 3 - Add an existing user to a new group

1. Sign in to the [Microsoft Entra Admin Center](https://entra.microsoft.com)

2. Select Groups > All groups > New Groups

3. Select the `Project23` group.

4. Select Members >  + Add members from the menu near the top.
