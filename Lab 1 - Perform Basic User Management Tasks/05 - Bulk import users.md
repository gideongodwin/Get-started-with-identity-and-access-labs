## Task 5 - Bulk import users

1. Sign in to the [Microsoft Entra Admin Center](https://entra.microsoft.com)

2. Select Users > All users

3. Select the Bulk operations and then select Bulk create.

   <img width="1398" height="288" alt="517729234-d6ca2a46-31fa-4420-b46b-3c95f5bd45d3" src="https://github.com/user-attachments/assets/c8a5f602-f6b0-4ea7-803a-3c2a91b8f1df" />
 
   > Selecting Bulk create will open a new tile. This tile provides a Download link to a template file that you will edit to populate with your user information and upload to add the bulk creation of users.

4. In the upload section, click the blue folder icon next to the CSV field.

5. Browse to your CSV file and open .

6. Wait for the notification that the file uploaded successfully.

7. Select the Submit button to add the users.
